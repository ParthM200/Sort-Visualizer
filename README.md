# Sort Visualizer

An animated side-by-side visualization of **Bubble Sort** and **Insertion Sort**, built in Java.

## Run

```bash
./run.sh
```

Requires JDK 11+.

## How it works

1. Set your array size and animation speed on the welcome screen
2. Hit **Visualize** — both algorithms start simultaneously on the same array
3. Watch comparisons (red) and sorted elements (green) animate in real time
4. Live stats track comparisons, swaps, and elapsed time per algorithm

## Controls

| Button | Action |
|---|---|
| Restart | Re-run both algorithms on the same array |
| New Array | Generate a new random array and re-run |
| ← Settings | Go back to the welcome screen |
